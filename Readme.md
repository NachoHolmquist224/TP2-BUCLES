# Trabajo Práctico: Estructuras de Control en JavaScript

Este proyecto consiste en una serie de ejercicios diseñados para poner en práctica conceptos fundamentales de **JavaScript**, enfocándose específicamente en la gestión de flujos de datos mediante estructuras condicionales y ciclos de repetición.

## 🚀 Objetivo del Proyecto

El propósito de este repositorio es demostrar la capacidad de resolver problemas algorítmicos de nivel básicos, medio y avanzado con validación de datos y manipulación de variables utilizando la sintaxis moderna de JS.

## 🛠️ Conceptos Aplicados

En el desarrollo de estos ejercicios se implementaron:

* **Estructuras Condicionales:** Uso de `if`, `else if`, `else` y `switch` para la toma de decisiones lógica.
* **Bucles e Iteraciones:** Implementación de ciclos `for`, `while` y `do-while` para el procesamiento de tareas repetitivas.
* **Operadores Lógicos:** Utilización de `&&` (AND), `||` (OR) y `!` (NOT) para condiciones complejas.
* **Entrada/Salida de Datos:** Interacción mediante `console.log()` y captación de datos (según el entorno del ejercicio).
